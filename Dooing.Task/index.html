About Dooing.Task

Tracking goals and staying productive shouldn't be a task in itself. So we wanted to create a task manager that keeps "you" in the moment. No stat tracking, no task labeling or priority to focus on except for "What you want to do".

Dooing.Task is based on 3 principles:

  • Planning: Set manageable goals and tasks for yourself by breaking down tasks into smaller, actionable steps. 3 Categories "Today" "Tomorrow" "Future". 6 Tasks Per Day = More Tasks Completed! Manageable and attainable.
  • Attention: Focus on one task at a time. Use the Pomodoro technique to work in focused sprints with breaks in between.
  • Retention: Reflect on your progress and achievements. Use the "Completed" section to see what you've accomplished along with the "Inspiration" section to keep you motivated. Save notes, study summaries or other things you want to remember or be inspired by!

Help & Tutorial

Welcome to Dooing.Task! Here's a quick guide to get you started:

  • Task Management: Add tasks for "Today," "Tomorrow," or a "Future" date. Use the arrow buttons in the header or your keyboard's left/right arrow keys to navigate between these main task categories.
  • Task Cards: Each task is displayed on a card. You can mark tasks as complete using the checkbox, edit task details, or delete tasks directly from the card.
  • Future View: In the "Future" tasks category, you can switch between a weekly and a monthly calendar view to see your upcoming tasks.
  • Pomodoro Timer: Access the Pomodoro timer via the sidebar to work in focused intervals. Customize session durations in the Pomodoro section itself or basic settings here.
  • Inspiration (Inspo): Find a daily dose of motivation with random quotes in the Inspo section, also accessible from the sidebar.
  • Settings: Change the application theme and configure other preferences in the Settings section.
  • Navigation: Use the sidebar (hamburger menu) to easily jump between different sections of the application like Tasks, Pomodoro, Inspo, About, Contact, Settings, and Help.

More detailed tutorials, FAQs, and keyboard shortcut guides will be added soon. If you have any questions, feel free to reach out via the Contact section!
